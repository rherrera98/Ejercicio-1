Git y Hub

Concepto de Git

Git es un sistema de control de versiones. Esto significa que cada desarrollador tiene una copia del historial del proyecto en su equipo local y no dependen de un servidor para trabajar.

Concepto de GitHub

GitHub es una plataforma en la web que permite almacenar y gestionar proyectos que utilizan Git como sistema de control de versiones. Fue lanzada en 2008 y adquirida por Microsoft en 2008 es la plataforma mas popular en el mundo para alojar codigo, colaborar en proyectos de software y gestionar versiones de manera remota.
